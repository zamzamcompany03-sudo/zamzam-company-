## 💰 معايير الأسعار والحسابات

### الذهب (عيار 21)

1. **مبلغ الذهب** = الوزن × سعر الغرام (من الرابط)
1. **المبلغ الكلي** = مبلغ الذهب + الأجور الثابتة
1. **الخصم** = المبلغ الكلي × (نسبة الخصم / 100)
1. **السعر النهائي** = المبلغ الكلي - الخصم

مثال:

```
الوزن: 5.52 جرام
سعر الغرام: 8,500 د.ع
مبلغ الذهب: 5.52 × 8,500 = 46,920 د.ع
الأجور: 15,000 د.ع
المبلغ الكلي: 46,920 + 15,000 = 61,920 د.ع
الخصم (10%): 61,920 × 0.1 = 6,192 د.ع
السعر النهائي: 61,920 - 6,192 = 55,728 د.ع
```

### الساعات والفضيات

1. **المبلغ الكلي** = نكتبه مباشرة (للساعات)
1. **المبلغ الكلي** = الوزن × سعر الغرام (للفضيات)
1. **الخصم** = المبلغ الكلي × (نسبة الخصم / 100)
1. **السعر النهائي** = المبلغ الكلي - الخصم
